# Support

- Use [GitHub Discussions](https://github.com/syb-illin/LogicLyrics/discussions) for setup questions, workflows and ideas.
- Use the [bug report form](https://github.com/syb-illin/LogicLyrics/issues/new/choose) for reproducible defects.
- Use the feature form for a scoped enhancement with a clear musician workflow.
- Follow [SECURITY.md](SECURITY.md) for vulnerabilities.

Before posting, remove lyrics, audio, artist names, project names, full paths and other private content. Include the Logic Lyrics version/build, macOS version and Logic Pro version when relevant.
